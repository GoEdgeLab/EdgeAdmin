源站协议
源站地址

源站服务器地址，通常是一个IP（或域名）加端口，不需要加 http:// 或 https://。
权重

数字越大，代表分配的请求比例越多。
名称

给当前源站起一个容易识别的名称。
连接失败超时时间
秒

连接源站失败的最大超时时间，0表示不限制。
读取超时时间
秒

读取内容时的最大超时时间，0表示不限制。
最大并发连接数

源站可以接受到的最大并发连接数，0表示使用系统默认。
最大空闲连接数

当没有请求时，源站保持等待的最大空闲连接数量，0表示使用系统默认。
最大空闲超时时间
秒

源站保持等待的空闲超时时间，0表示使用默认时间。
