路径匹配规则 *

路径通常以斜杠(/)开头，比如/hello。如果匹配类型是正则表达式匹配，则可以是一个正则表达式
匹配类型

终止往下匹配

如果选中了此选项，一旦匹配成功，不会继续匹配其他的路由规则。
名称

可以用来说明此规则用途。。
匹配条件
专属域名

默认不需要填写，表示支持所有域名。如果填写了专属域名，表示当前路由规则只会在所列的专属域名被访问时才生效。
不区分大小写

选中表示匹配规则中的路径中的英文字母不区分大小写。
反向匹配

选中表示匹配所有不符合规则的路径。
描述
启用当前路由规则
