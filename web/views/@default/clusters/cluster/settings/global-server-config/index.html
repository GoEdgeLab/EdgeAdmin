域名

禁止未绑定域名访问

选中后，表示禁止未绑定的域名和IP访问。
页面提示

访问未绑定的域名时提示的文字，可以使用HTML；仅限于HTTP请求，不适于用HTTPS（HTTPS会提示证书错误）。
允许例外的域名

允许这些域名不经过绑定就可以直接访问网站。
允许使用节点IP访问

选中后，表示允许使用节点IP访问网站。
默认域名

例外域名或使用节点IP访问时使用的默认域名。

访问日志

只记录通用请求
Header

选中后，表示访问日志中只记录通用的HTTP请求Header（比如User-Agent），其他自定义或非标准的（比如Test-Header）将不记录。

运行日志

记录服务错误

在节点运行日志中记录网站服务相关错误，比如无法连接源站等。
